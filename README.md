# Онлайн-библиотека

Курсовой проект на тему "Разработка веб-интерфейса для онлайн-библиотеки".

Проект выполнен на базовом стеке фронтенд-технологий:

- HTML
- CSS
- JavaScript

## Описание

Веб-интерфейс позволяет просматривать каталог книг, искать книги по названию или автору, фильтровать список по жанру, добавлять книги в пользовательский список "Мои книги", работать с избранным, а также открывать небольшую карточку с описанием книги и выставлять оценку.

## Основные функции

- отображение каталога книг;
- поиск по названию и автору;
- фильтрация по жанру через выпадающий список и кнопки жанров;
- переключение между вкладками "Все книги" и "Избранное";
- добавление книг в избранное;
- добавление книг в список "Мои книги";
- удаление книг из выбранного списка;
- просмотр описания книги в отдельной карточке;
- выставление оценки книге от 1 до 5;
- добавление новой книги через форму;
- редактирование описания пользовательской книги;
- удаление пользовательской книги из каталога;
- переключение светлой и темной темы.

## Назначение проекта

Проект предназначен для демонстрации базовых навыков разработки клиентского веб-интерфейса: семантической HTML-разметки, адаптивной CSS-верстки, работы с DOM, обработки событий и управления данными на стороне браузера.
